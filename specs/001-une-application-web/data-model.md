# Phase 1 : Data Model

## Entités principales

### Utilisateur
- Identifiant (anonyme/local)
- Langue maternelle
- Niveau (débutant, intermédiaire, avancé)

### Texte
- Contenu (string)
- Langue détectée
- Longueur (nombre de mots)

### Lecture
- Langue utilisée
- Voix (masculine/féminine)
- État (en cours, terminé, erreur)
